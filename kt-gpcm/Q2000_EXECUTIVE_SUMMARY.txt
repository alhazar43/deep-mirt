EXECUTIVE SUMMARY: Q=2000 ALPHA RECOVERY DIAGNOSIS
================================================================================

PROBLEM:
--------
Alpha recovery degrades 57% from Q=1000 (r_α=0.644) to Q=2000 (r_α=0.275).
Paradoxically, Q=5000 achieves r_α=0.654, better than Q=2000 despite being
2.5x larger.

ROOT CAUSE:
-----------
Q=2000 dataset crosses a CRITICAL THRESHOLD for parameter identifiability:

  Q=2000 (BELOW THRESHOLD):
    - 3,000 students (40% fewer than Q≤1000)
    - 127.5 responses/item (49% fewer than Q=1000)
    - 4.25% item coverage per student
    - Interaction density: 382k (students × responses/item)
    → r_α = 0.275-0.351 (FAILED)

  Q=1000 (ABOVE THRESHOLD):
    - 5,000 students
    - 250 responses/item
    - 5.0% item coverage per student
    - Interaction density: 1,250k
    → r_α = 0.644 (SUCCESS)

  Q=5000 (ABOVE THRESHOLD):
    - 1,000 students (fewer than Q=2000!)
    - 100 responses/item (fewer than Q=2000!)
    - 10.0% item coverage per student (2.4x Q=2000)
    - Long sequences (avg 500 vs 85) provide rich θ estimates
    → r_α = 0.654 (SUCCESS)

KEY INSIGHT:
------------
The model needs sufficient student-item interaction density to disentangle:
  1. Student ability (θ) - requires long sequences per student
  2. Item discrimination (α) - requires diverse students per item
  3. Item difficulty (β) - requires diverse students per item

Q=2000 fails because BOTH student count AND responses/item fall below
critical values simultaneously. Q=5000 succeeds because very long sequences
(avg 500) compensate for fewer students by providing:
  - Rich θ estimates per student
  - 2.4x better item coverage per student
  - Stronger gradient signal for item parameters

EVIDENCE:
---------
✓ Model architecture: IDENTICAL across all Q values
✓ Hyperparameters: IDENTICAL across all Q values
✓ Training convergence: Both Q=1000 and Q=2000 achieve QWK ≈ 0.75
✓ True alpha distributions: CONSISTENT across all datasets
✓ Training curves: Both models converge smoothly

→ The issue is purely DATASET-DRIVEN, not model or optimization failure.

RECOMMENDATION:
---------------
REGENERATE Q=2000 dataset with:

  n_students: 5000 (match Q≤1000)
  seq_len_range: [100, 200] (2-4x longer)

Expected outcomes:
  - Responses/item: ~750 (6x current)
  - Coverage/student: 7.5% (1.8x current)
  - Interaction density: 3,750k (10x current)
  - Predicted r_α: 0.65-0.70

Alternative (if computational budget allows):
  n_students: 2000
  seq_len_range: [200, 400]
  Predicted r_α: 0.70-0.75

NOT RECOMMENDED:
  - Training longer (marginal improvement, won't solve data sparsity)
  - Larger batch size (won't solve fundamental issue)
  - Adding regularization (may bias recovery metrics)
